# Intelligent Catalogue Builder

## Overview
An AI-powered retail analytics dashboard that converts sales reports into digital catalogues and marketing content.

## Features
- Sales report parsing
- Product catalogue generation
- Brand detection
- Pack size detection
- Category and subcategory detection
- Margin calculation
- Duplicate removal
- Product image finder
- Recommended image selection
- Marketing creative generation
- Push notifications
- WhatsApp marketing messages
- Campaign headlines
- Catalogue export

## Technologies Used
- Python
- Streamlit
- Pandas
- Matplotlib

## How to Run

pip install -r requirements.txt

streamlit run app.py
